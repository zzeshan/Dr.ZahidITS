## Publish on GitHub Pages
1. Upload all files to the root of your public repository.
2. Go to **Settings > Pages**.
3. Choose **Deploy from branch**.
4. Select **main** and **/(root)**.
5. Save and wait for deployment.

## To improve Google indexing
- Add your site to Google Search Console.
- Request indexing for the homepage and other pages.
- Keep page titles and content descriptive and public.
